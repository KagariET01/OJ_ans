# CF 1842C Tenzing and Balls 題解
~~他寫要用DP解，那答案一定是DP~~  
建一DP[n+1]  
DP[i]代表前綴i(意即a1~ai)的最佳解法中，會留下多少個字母  
例如a={1,2,2,3,3}  
DP={1,2,1,2,1}  
很明顯，轉移式為DP[i]=min(DP[i-1]+1,DP[j]) (a[j+1]=a[i])  
DP[i-1]+1是不做任何刪除，單純把a[i]放到最後面的答案  
DP\[j](a[j+1]=a[i])是把a[i]到a[j+1]刪掉後剩下的字串長度
